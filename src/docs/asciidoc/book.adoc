== HTTP verbs

- 해당 API 는 도서 관리를 위한 API 명세서 입니다.

== 요청 메소드

|===
| Verb | Usage

| `GET`
| 조회 요청은 GET Method 를 사용 하세요


| `POST`
| 등록 요청은 POST Method 를 사용 하세요

| `PUT`
| 수정 요청은 PUT Method 를 사용 하세요

| `DELETE`
| 우리 서비스는 DELETE 요청을 사용하지 않습니다.
|===

[[overview_http_status_codes]]
== HTTP status codes

해당 API 는 RESTful API 으로 가능한 한 표준 HTTP 및 REST 규칙을 준수했습니다.

|===
| Status code | Usage

| `200 OK`
| 요청 성공 시 응답 코드

| `201 Created`
| 등록 요청 성공 시 응답 코드

| `400 Bad Request`
| 요청에 문제가 있을 시 응답 코드

| `404 Not Found`
| 요청 URL 이 잘못된 경우 응답 코드
|===
